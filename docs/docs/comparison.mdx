Accountant24 sits between the two main camps. It uses the same data model as hledger — plain-text, double-entry bookkeeping — but you don't need to learn the syntax. The experience is closer to YNAB, Monarch, or Actual Budget, except your data stays on your machine, there's no subscription, and the interface is a desktop app you chat with.
